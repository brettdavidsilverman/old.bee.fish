True Functions

Literary example

  • A machine that builds a new machine with the same purpose has a true function.

Mathematical example

  • If the function f is a function of itself
  • Then f = ff is a true function.

Physics example

  • The coulomb Cu is a measure of energy as is Einsteins E = mC2.
  • Combining them Cu = bmC2
  • Making CuB = fBC2 a true function.
