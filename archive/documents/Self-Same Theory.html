Self-Same theory

Literary example

  • 1: Starting point: you have a machine that builds a new machine with the same function.
  • 2: Build a new machine from the starting machine.
  • 3: Build a second machine from the last machine.
  • 4: Repeat step 3 until the earth runs out of resources.
  • Question: "How does a light bulb work?"
  • Answer: "Switch it on."

The only time you can prove it works is after you've run it. Obviously this is correct as every light bulb breaks eventually.

If we can presume the following we can prove the machines function without having to consume infinite resources.

  • 5: Part of the machines function is to reset at the end of its cycle. (We need this for step 8)
  • 6: None of the inputs to the machine have changed.
  • 7: All of the machines function is used.
  • 8: The last and second last machines are equivalent.

You have now proven all of the machines function.
